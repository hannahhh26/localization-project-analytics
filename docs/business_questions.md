# Business Questions

Questions we aim to answer through data visulaization.

## Project Performance

- Which projects are overdue and is there any common factors?
- What is the average turnaround time by project type and client?
- Which project types require the most effort?
- Which project types come with the most red flags?

## Project Manager Insights

- How many clients on average does each PM work on
- How is workload distributed across project managers?
- Are there areas where only one PM has knowledge of a client or workflow?
- How much cover exists for holidays or unexpected absence?

## Client Insights

- Which clients generate the highest volume of projects?
- Which clients generate the highest revenue?
- What types of project do clients give us? (e.g. marketing materials, apps, UI, eLearning etc.)

## Translator Performance

- Which translators have the highest assessment score?
- Which translators consistently deliver late?
- Which languages have insufficient translator coverage to meet demand?
- Which translators work across multiple clients?

## Financial & Operational Insights

- Which project types provide the best return on effort?
- How does project revenue compare against delivery effort?
- Which projects consume the most PM, translator, and QA time?
